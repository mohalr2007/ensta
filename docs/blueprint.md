# **App Name**: Polyglot School Hub

## Core Features:

- Home Page: Display school information, announcements, and featured content.
- Contact Page: Provide contact information for the school, including a contact form.
- About Page: Showcase school history, mission, and values.
- Language Switching: Allow users to switch between French and English.
- Theme Switching: Implement light and dark themes for user preference.

## Style Guidelines:

- Primary color: Rouge vif / bordeaux (#D32F2F or #B71C1C) for accents, buttons, and titles.
- Secondary color: Violet profond / indigo (#4527A0 or #311B92) for backgrounds, navigation bars, and graphic elements.
- Background color: Blanc pur (#FFFFFF) for text on dark backgrounds or light backgrounds.
- Text color: Noir élégant (#000000 or #212121) for titles, subtitles, and main text on light backgrounds.
- Body and headline font:  Use a modern, readable font for both headlines and body text.
- Use simple, clear icons for navigation and key features.
- Clean and responsive layout for optimal viewing on all devices.
- Subtle transitions when switching languages and themes.
